# Instalação

A instalação do ErionImpressos é simples e rápida. Siga os passos abaixo.

1. Acesse a pasta `\\anderson-pc\COMPARTILHAMENTO\XML CLIENTES\XML Rafael\ErionImpressos`
2. Execute o instalador `ErionImpressos_Setup_X.X.X.exe`
3. Após iniciar o instalador, basta ir clicando em avançar.

::: tip IMPORTANTE
Certifique-se de marcar a opção **"Criar ícone na Área de trabalho"** durante o processo.
:::

### Requisitos do Sistema

Antes de iniciar a instalação, verifique se o computador é Windows 10 ou superior, pois o sistema não possui suporte para Windows 7 ou 8.
