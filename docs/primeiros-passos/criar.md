# Criar Etiqueta

Para iniciar um projeto do zero, basta clicar no botão `+` (**Criar**), localizado no topo da barra de navegação lateral esquerda. 

Ao clicar, um modal centralizado será exibido com duas abordagens de criação: a seleção de tamanhos de etiquetas prontos ou a personalização manual das dimensões.

## 1. Aba: Tamanhos Prontos

Esta é a tela inicial do modal. Ela oferece uma grade de cards com as dimensões de etiquetas prontas para utilizar, agilizando o processo de abertura do editor.

![Aba Tamanhos Prontos](/assets/criar_etiqueta_prontos.png)

* **Barra de Busca Interna:** No topo do painel, o campo *"Pesquise pelo tamanho"* permite filtrar rapidamente a lista digitando números correspondentes à largura ou altura desejada.
* **Grade de Padrões:** Cada card exibe graficamente a proporção física da etiqueta e suas dimensões exatas em milímetros (Largura x Altura).

::: tip 💡 Como iniciar usando um padrão
Basta dar **um clique sobre o card** com o tamanho desejado. O sistema configurará as dimensões e redirecionará direto para o **Editor**.
:::

---

## 2. Aba: Personalizar

Caso a sua etiqueta possua medidas específicas que não constam na listagem padrão, mude para a aba **Personalizar** no menu esquerdo do modal.

![Aba Personalizar Tamanho](/assets/criar_etiqueta_personalizar.png)

Neste painel, você define os parâmetros exatos da etiqueta:

* **Largura e Altura:** Campos numéricos para digitação manual do tamanho em milímetros (mm).
* **Dimensões Recentes:** Logo abaixo dos campos de inserção, o sistema exibe uma lista dinâmica com as últimas medidas customizadas criadas, permitindo reaproveitá-las com apenas um clique.

::: warning ⚠️ Limitações
O editor trabalha com tamanhos de etiqueta que variam de **10 mm até 300 mm** tanto para largura quanto para altura. Valores fora desse intervalo não serão aceitos pelo campo.
:::

Após preencher as medidas desejadas, clique no botão **Criar novo design** para abrir a área de edição.
